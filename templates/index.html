What is Deepfake?

Deepfakes are synthetic media where a person in an existing image or video is replaced with someone else's likeness using artificial intelligence techniques.

How to Protect Yourself from Deepfakes?

  • Be cautious about sharing personal videos and images online.
  • Use trusted sources to verify content authenticity.
  • Report suspected deepfake content to appropriate platforms.
  • Use deepfake detection tools if available.
